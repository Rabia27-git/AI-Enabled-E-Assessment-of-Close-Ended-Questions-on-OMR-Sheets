# AI-Enabled E-Assessment of Close-Ended Questions on OMR Sheets

## Overview

This project is an AI-powered Optical Mark Recognition (OMR) evaluation system designed to automatically assess multiple-choice question (MCQ) answer sheets.

## Features

- Supports OMR sheets containing **100, 180, and 200 MCQs**
- AI-assisted answer sheet evaluation with high accuracy
- Generates detailed results and psychometric reports
- Designed for real-world examination and assessment requirements
- Currently deployed and actively used by SIBA Testing Services

## Setup Instructions

- Before running the project, install all required Python libraries and dependencies.
- Open File.py.
- Navigate to line 65 and update the path to the folder containing scanned OMR answer sheet images.
- Navigate to line 76 and replace the placeholder with your OpenRouter API key.
- Save the file and run the project.

## Notes

- Ensure the input folder contains only valid OMR sheet images.
- The dummy answer key is already included in the project folder.
- Update the answer key to match your actual exam.
